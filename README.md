# Four-class-Lung-CT-Model
本研究针对肺部疾病临床诊断中人工读片效率低、 主观性强的问题， 提出一种基于多任务学习的四分类肺 部 CT 模型， 实现对 COVID-19、 肺不透明（Lung_Opacity）、 正常（Normal） 和病毒性肺炎（Viral Pneumonia） 的精准分类， 并生成肺部区域掩膜辅助可视化诊断。 通过以 U-Net++ 作为主干网络， 结合分类与分割任务联合 优化， 在小样本（每类 30 例） 和完整数据模式下验证模型性能。 实验结果表明， 模型在小样本模式下分类准确 率达 91.67%， 完整数据模式下提升至 96.36%， 掩膜生成的 Dice 系数分别为 0.6401 和 0.6611， 为肺部疾病的 计算机辅助诊断提供了有效解决方案。
